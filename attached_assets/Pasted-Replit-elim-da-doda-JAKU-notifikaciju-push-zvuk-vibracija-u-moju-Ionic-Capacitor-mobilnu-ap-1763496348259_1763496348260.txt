Replit, želim da dodaš JAKU notifikaciju (push + zvuk + vibracija) u moju Ionic/Capacitor mobilnu aplikaciju za praćenje reklamacija. Molim te da uradiš sledeće korake AUTOMATSKI:

────────────────────────────────────────
### 1. Instaliraj potrebne Capacitor pluginove

Pokreni:
npm install @capacitor/push-notifications
npm install @capacitor/local-notifications
npm install @capacitor/haptics
npm install @capacitor/badge
npx cap sync

────────────────────────────────────────
### 2. U fajl `client/src/main.ts` (ili glavni entry point) dodaj ovaj kod:

import { PushNotifications } from '@capacitor/push-notifications';
import { LocalNotifications } from '@capacitor/local-notifications';
import { Haptics, ImpactStyle, NotificationHapticsStyle } from '@capacitor/haptics';
import { Badge } from '@capacitor/badge';

//// PRIJAVA NA PUSH
PushNotifications.requestPermissions().then(result => {
  if (result.receive === 'granted') {
    PushNotifications.register();
  }
});

//// PUSH EVENT — kada stigne nova reklamacija od servera
PushNotifications.addListener('pushNotificationReceived', async (notification) => {
  playAlertSound();
  vibrateStrong();
  showLocal(notification.title, notification.body);
});

//// JAKA VIBRACIJA
async function vibrateStrong() {
  await Haptics.notification({ type: NotificationHapticsStyle.ERROR });
  await Haptics.impact({ style: ImpactStyle.Heavy });
}

//// PUŠTANJE ZVUČNOG ALARMA
async function playAlertSound() {
  await LocalNotifications.schedule({
    notifications: [
      {
        id: Date.now(),
        title: "Nova reklamacija!",
        body: "Otvori aplikaciju.",
        sound: "alert1",
        autoCancel: true
      }
    ]
  });
}

//// LOKALNA POPUP NOTIFIKACIJA
async function showLocal(title, body) {
  await LocalNotifications.schedule({
    notifications: [
      {
        id: Date.now(),
        title,
        body,
        sound: "alert1",
        smallIcon: "ic_stat_icon"
      }
    ]
  });
}

────────────────────────────────────────
### 3. Dodaj jak zvuk za notifikaciju

Kreiraj fajl:
android/app/src/main/res/raw/alert1.mp3

(Replit može da ga kreira kao prazan – ja ću ručno uploadovati pravi zvuk kasnije.)

────────────────────────────────────────
### 4. Android permissions

U fajl:
android/app/src/main/AndroidManifest.xml

Dodaj:

<uses-permission android:name="android.permission.VIBRATE" />
<uses-permission android:name="android.permission.POST_NOTIFICATIONS" />
<uses-permission android:name="android.permission.WAKE_LOCK" />

────────────────────────────────────────
### 5. Badge broj na ikoni (opciono)

Poziv:
Badge.set({ count: 1 });
